搭建Hexo项目

发表于2023-01-14|更新于2023-01-14|静态博客搭建教程

搭建Hexo项目

Hexo框架 简介:

  • Hexo是一个使用前端技术开发的博客框架.

搭建前准备:

  • 1.Hexo依赖于Node.js,所以要求必须已经安装了新版的Node.js,并且已经将Node.js配置在环境变量中了
  • 2.Hexo需要从GitHub拉取项目,所以要求已经安装了Git,并且已经Git配置在环境变量中了

  • 只有满足上面两个前提,才能开始下面的搭建步骤,否则必定搭建失败.

搭建Hexo博客项目

  • 1.下载搭建Hexo项目的脚手架

    • sudo npm install hexo-cli -g

  • 2.进入要搭建博客项目的目录中

    • # 我要在 ~/Documents 这个目录中搭建Hexo项目
cd  ~/Documents

  • 3.创建Hexo项目

    • # 我这里的创建一个名叫 JohnBlog 的Hexo博客项目
hexo init JohnBlog

  • 4.进入项目目录中

    • cd JohnBlog

  • 5.安装项目依赖库

    • pnpm install

  • 6.启动项目

    • hexo server  # 可以简写为 hexo s

  • 7.浏览器访问博客服务地址

    • http://localhost:4000

    • 如果能访问成功,那么一个简单的Hexo博客项目就搭建好了

  • 8.将自己的Markdown文件(或文件夹)复制到放到source/_posts目录中

    • 略

  • 9.将图片复制到source/img目录中

    • source/img目录用于存放

  • 9.清除已编译的编译静态文件、重新编译静态文件、开启服务

    • ```shell
      hexo clean & hexo g & hexo server

安装 代码高亮插件 (必须)

  • 说明:

    • Hexo默认自带的代码高亮插件不支持我写的Markdown笔记格式: 我的代码块写在无序列表中,而Hexo自带的代码插件无法正确识别,此时就需要一个插件来正确识别我的笔记格式

  • 操作步骤:

    • 1.下载插件
      • pnpm i -S hexo-prism-plugin
    • 2.修改 Hexo博客根目录下的_config.yml文件
      • highlight:
  # 设置为false(关闭)
  enable: false
prismjs:
  # 设置为false(关闭) 
  enable: false

# 采用的代码高亮插件
prism_plugin:
  # 可选值: realtime/preprocess
  mode: 'preprocess'    
  theme: 'tomorrow'
  # 代码是否显式 行号 (默认为false表示不显示)
  line_number: false
  custom_css:
    • 3.重新编译并重启项目
      • hexo clean & hexo g & hexo server
