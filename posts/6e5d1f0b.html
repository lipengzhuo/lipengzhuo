Hexo项目目录结构介绍

Hexo项目根目录

  • Hexo项目的文件结构图

    • ![](/Users/johnmarx/Library/Mobile Documents/comappleCloudDocs/JohnNodes/笔记插图/静态插图/20210416221656449.png)

  • scaffolds目录: 模板文件夹

    • 当您新建文章时,Hexo会根据scaffold来建立文件

    • scaffolds目录中的.md文件都是各个模板.

      • page.md是我们新建页面的模板
      • post.md是我们新建博客的模板
    • 我们可以修改这些默认的模板

  • source目录: 工作目录

    • _posts目录: 存放博客文章(Markdown文件) (我们就是在此目录中写博客的)

    • 除_posts文件夹之外,开头命名为_(下划线)的文件或文件夹和隐藏的文件将会被忽略.

    • Markdown和HTML文件会被解析并放到public文件夹,而其他文件会被拷贝过去.

  • themmes目录: 主题文件夹

    • themmes目录中专门用于存放 博客的主题

    • Hexo 会根据主题来生成静态页面

  • _config.yml: 项目配置文件

    • _config.yml文件是Hexo项目的主配置文件,项目的大部分配置都在此文件中

  • package.json:

    • 记录当前项目的信息: 包括当前先项目版本、依赖库 等信息

配置文件_config.yml详解:

  • # 网站大标题
title: 清玄门徒
# 网站小标题
subtitle: '清玄门徒'
# 网站描述
description: 专注于Web,分享知识
# 网站关键词
keywords: [HTML, CSS, JavaScript, JQuery, React, Vue.js]
# 网站作者
author: John Lee
# 博客使用的语言 (en是英语 zh-CN是中文)
language: zh-CN
# 时区(可以不写)
timezone: ''

# 博客dURL
url: http://example.com
permalink: :year/:month/:day/:title/
permalink_defaults:
pretty_urls:
  trailing_index: true # Set to false to remove trailing 'index.html' from permalinks
  trailing_html: true # Set to false to remove trailing '.html' from permalinks


# 项目目录
source_dir: source
public_dir: public
tag_dir: tags
archive_dir: archives
category_dir: categories
code_dir: downloads/code
i18n_dir: :lang
skip_render:

# Writing
new_post_name: :title.md # File name of new posts
default_layout: post
titlecase: false # Transform title into titlecase
external_link:
  enable: true # Open external links in new tab
  field: site # Apply to the whole site
  exclude: ''
filename_case: 0
render_drafts: false
post_asset_folder: false
relative_link: false
future: true
# 代码 高亮显式方式 
highlight:
  enable: true
  line_number: true
  auto_detect: false
  tab_replace: ''
  wrap: true
  hljs: false
prismjs:
  enable: false
  preprocess: true
  line_number: true
  tab_replace: ''

# Home page setting
index_generator:
  path: ''
  per_page: 10
  order_by: -date

# Category & Tag
default_category: uncategorized
category_map:
tag_map:

# Metadata elements
meta_generator: true

# Date / Time format
date_format: YYYY-MM-DD
time_format: HH:mm:ss
## updated_option supports 'mtime', 'date', 'empty'
updated_option: 'mtime'

# Pagination
per_page: 10
pagination_dir: page

# Include / Exclude file(s)
include:
exclude:
ignore:

# 博客使用的主题 (landscape是自带的主题)
theme: landscape

# 将项目部署到远程仓库中
deploy:
  type: git
  # 将repo参数的地址 换成 第一步中创建的Gitee仓库地址
  repo: https://gitee.com/lipengzhuo/lipengzhuo.git
  # 分支名称
  branch: master
